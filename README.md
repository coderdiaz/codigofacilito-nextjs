## Template Next.js

A Next.js template with TypeScript and TailwindCSS configured.

### Configured tools

- TailwindCSS.
- TypeScript.
- EsLint.
- Prettier.

### How to use

1. Clone this repository
2. Run `yarn install` or `npm install`.
3. Run `yarn dev` or `npm run dev`.
4. Open your browser to see the result.
